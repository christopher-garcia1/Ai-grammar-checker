Ai Grammar Checker
=================

Overview
--------
Ai Grammar Checker is a web-based application that leverages artificial intelligence to analyze and improve the grammar of input text. Built with Node.js and EJS, this tool provides real-time grammar suggestions to enhance writing quality.

Features
--------
- Real-time Grammar Checking: Instantly analyzes text for grammatical errors and provides suggestions.
- User-Friendly Interface: Simple and intuitive design for seamless user experience.
- Built with Node.js and EJS: Utilizes server-side JavaScript and embedded JavaScript templates for dynamic content rendering.

Installation
------------
Prerequisites:
- Node.js (version 14 or higher)

Steps:
1. Clone the repository:
   git clone https://github.com/christopher-garcia1/Ai-grammar-checker.git
   cd Ai-grammar-checker

2. Install dependencies:
   npm install

3. Create a .env file in the root of the project with your OpenAI API key:
   OPENAI_KEY='Your key'
   (Replace 'Your key' with your actual OpenAI API key)

4. Start the application:
   npm start

5. Open your browser and navigate to http://localhost:3000 to use the application.

Usage
-----
- Enter or paste text into the provided input field.
- Click the "Check Grammar" button to analyze the text.
- Review the suggested corrections and apply them as needed.
